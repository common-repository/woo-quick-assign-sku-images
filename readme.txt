== Description ==

Looking for a tool to add image bunch for your WooCommerce products? Do not waste your time assigning them one by one. Try Quick Assign Products Images by SKU Plugin – it will help you easy and accurately assign images  to your products by product SKU.

== Frequently Asked Questions ==

Quick Assign Products Images by SKU for WooCommerce is an easy-to-use and intuitive tool which will assign images for your WooCommerce products automatically.

To use the plugin navigate to Admin panel > WooCommerce > Quick Assign Products Images by SKU:

How to use the plugin:

1. Create a product in WooCommerce and copy its SKU ID;
2. Prepare an image you wish to add and rename this image on your local system – paste saved SKU (step#1);
3. If product SKU contains forbidden symbols, please replace them using the allowed ones (special table is displayed on plugin setting page) ;
4. Open the plugin page. Select image file and press upload button;
5. Check out your product in WooCommerce.

Result: your product image was added/updated.

The plugin is developed and supported by <a href="https://www.extreme-idea.com/">Extreme Idea LLC</a>. Our entire team is ready to help you. Ask your questions in the support forum, or <a href="https://www.extreme-idea.com/contact-us/">contact us directly</a>.

== Installation ==

Installation

1. Log in as an administrator to WordPress Admin Panel.
2. Navigate to Plugins > press Add New button > press Upload Plugin button.
3. Browse for plugin.zip archive > press Install Now button.
4. The plugin should be successfully installed. Activate the plugin.

== Uninstallation ==

To uninstall the plugin:

1. Log in to control panel as administrator.
2. Go to Plugins > Installed plugins > Woo Quick Assign SKU Images.
3. Deactivate the plugin.
4. Delete the plugin.

== Upgrade Notice ==

To update the plugin:

1. Log in as administrator to WordPress Admin Panel.
2. Navigate to Plugins > Installed Plugins > Woo Quick Assign SKU Images.
3. Press Deactivate button. Press Delete button. The plugin should be successfully uninstalled.
4. Navigate to Plugins > Add New > Upload Plugin.
5. Browse for 'Woo Quick Assign SKU Images'.zip file > press Install Now button.

The plugin should be successfully re-installed.
